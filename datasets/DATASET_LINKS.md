# Dataset Links

The final submission package does not duplicate the full datasets because the RoG graph-augmented parquet files are large. The code loads them directly from Hugging Face through `datasets.load_dataset(os.path.join(data_path, d), split=split)`.

## Primary Datasets

| Dataset | Link | Used for | Loading arguments |
|---|---|---|---|
| RoG-CWQ | https://huggingface.co/datasets/rmanluo/RoG-cwq | Main full-test KGQA benchmark and DVI/GraphLite experiments | `--data_path rmanluo --d RoG-cwq --split test` |
| RoG-WebQSP | https://huggingface.co/datasets/rmanluo/RoG-webqsp | Transfer/embedding-guided path-selection checks | `--data_path rmanluo --d RoG-webqsp --split test` |

## Local Data Policy

For Canvas/GitHub submission, this package includes source code, configuration files, report, poster, and compact result tables. It excludes generated `predictions.jsonl`, raw result directories, model checkpoints, and downloaded dataset parquet files. These can be regenerated by following the commands in `report/TriGCR_final_report.pdf` and the dataset links above.
